# Kata Source

This file lists useful links to original kata descriptions and resources for designing and practicing new katas. 
Use these sources to find requirements, inspiration, and exercises for your own kata sessions.

## Kata Sources

- [Kata-Log Rocks](https://kata-log.rocks/) — Large collection of katas and programming exercises
- [Coding Dojo Katas](https://codingdojo.org/kata/) — Extensive kata library for coding dojos
- [Codurance Katas](https://www.codurance.com/katas/) — Classic katas for software craftsmanship
- [CodeKata](http://codekata.com/) — Foundational programming katas and exercises
- [Advent of Code](https://adventofcode.com/) — Annual coding challenge with daily problems
- [Awesome Katas (gamontal)](https://github.com/gamontal/awesome-katas?tab=readme-ov-file) — Curated list of kata resources
- [TDD Katas (garora)](https://github.com/garora/TDD-Katas/tree/main/src) — TDD-focused kata implementations

## How to Use

- For each new exercise, create a dedicated folder in the repository to encapsulate the kata and its practice objective.
- When adding a new kata, include a link to its original source here if available.
- Feel free to expand this list as the repository grows.
